# Blockchain-transaction-explorer
A simple tool that allows users to explore blockchain transactions and wallet activity.

Features:
-Search transactions
-View wallet activity and transaction history 
-Monitor blockchain transfers
-Basic Web3 analytics for blockchain data

##Technologies
-Python 
-Blockchain APIs (Etherscan /Web3 APIs)
-JSON / REST API
-Web3 data integration

##Purpose
This project was created to explore blockchain data and understand how transaction and wallet activities can be monitored using Web3 technologies.

##Future Improvemntes
-Support for multiple blockchain networks
-Real-time transaction tracking
-Advanced blockchain analytics
-Visual dashboard for trasaction monitoring
